# Reference Audio / 参考音频

## English

Purpose: store the small reference voice sample used to document the DuplexOmni
agent voice.

Code role: this directory is an asset directory, not executable code.

Usage: use `google_Leda.wav` together with the reference text below when a TTS
or voice-cloning stage needs the public reference voice prompt.

```text
The weather is nice, and I speak calmly and clearly. 今天天气很好，我平静清楚地说话。
```
